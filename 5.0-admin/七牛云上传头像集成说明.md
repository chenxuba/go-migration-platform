# 七牛云上传头像集成说明

## 1. 安装七牛云SDK

在 `5.0-admin` 目录下执行：

```bash
pnpm add qiniu-js
```

## 2. 已完成的修改

### 2.1 创建API接口文件

文件：`src/api/qiniu.ts`

包含两个接口：
- `getQiniuToken()` - 获取图片上传token
- `getVideoUploadToken()` - 获取视频上传token

### 2.2 修改学员创建组件

文件：`src/components/common/create-student.vue`

**新增导入：**
```javascript
import * as qiniu from 'qiniu-js'

import { getQiniuToken } from '@/api/qiniu'
```

**新增功能：**
1. 头像上传状态管理 `uploadingAvatar`
2. 上传前校验函数 `beforeAvatarUpload`
3. 自定义上传处理函数 `handleAvatarUpload`

**UI改进：**
- 添加上传按钮loading状态
- 添加上传提示文字
- 优化布局样式

## 3. 使用说明

### 3.1 上传流程

1. 用户点击"上传学员头像"按钮
2. 选择图片文件（支持jpg、png，最大5MB）
3. 前端校验文件类型和大小
4. 调用后端接口获取七牛云上传token
5. 使用七牛云SDK直传文件到七牛云
6. 上传成功后更新头像URL

### 3.2 文件命名规则

上传的文件会保存为：`avatar/{uuid}{扩展名}`

例如：`avatar/132451e5-c153-4241-a234-284bfecd5698.jpg`

### 3.3 安全限制

后端已配置：
- 图片最大大小：10MB
- 允许类型：`image/*`
- 需要登录才能获取token

## 6. 常见问题

### Q1: 提示"获取上传凭证失败"
**A:** 检查是否已登录，确保token有效

### Q2: 上传成功但图片无法显示
**A:** 检查七牛云bucket域名配置是否正确

### Q3: 如何修改上传文件大小限制？
**A:** 
- 前端：修改 `beforeAvatarUpload` 函数中的 `isLt5M` 判断
- 后端：修改配置文件中的 `image.maxSize` 值

### Q4: 如何添加上传进度条？
**A:** 在 `handleAvatarUpload` 函数的 `observable.subscribe` 中的 `next` 回调里获取进度：

```javascript
next(res) {
  const percent = Math.floor(res.total.percent)
  console.log('上传进度:', percent + '%')
  // 可以在这里更新进度条UI
}
```
